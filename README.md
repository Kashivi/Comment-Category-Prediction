# Comment-Category-Prediction
Built a machine learning model to classify user comments into categories(Supervised learning)

• Performed data preprocessing including handling missing values, text cleaning, and feature extraction.  
• Applied feature engineering techniques to improve model performance.  
• Trained and evaluated multiple classification models using Scikit-learn.  
• Tuned hyperparameters and selected the best-performing model based on evaluation metrics.  
• Achieved 0.81 accuracy on validation data.  
• Analyzed model performance and identified key factors affecting prediction accuracy.
